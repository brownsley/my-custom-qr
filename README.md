# My Custom QR

QR Code များကို လွယ်ကူစွာ ဖန်တီးအသုံးပြုနိုင်ရန် ရည်ရွယ်ပါသည်။
Add this to your `pom.xml` to use this library:

```xml
    <repositories>
		<repository>
		    <id>jitpack.io</id>
		    <url>https://jitpack.io</url>
		</repository>
	</repositories>     

    <dependency>
	    <groupId>com.github.brownsley</groupId>
	    <artifactId>my-custom-qr</artifactId>
	    <version>1ab4cfdaa5</version>
	</dependency>
